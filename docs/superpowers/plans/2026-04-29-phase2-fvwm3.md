## Task 1: Passive button grabs (GrabButton / UngrabButton / AllowEvents)

WMs register passive grabs on frame windows with `GrabButton` so they receive
`ButtonPress` when a user clicks a decoration. Without this fvwm3 never receives
title-bar clicks and cannot move/resize/focus windows.

**Files:**
- Modify: `crates/yserver-core/src/server.rs`
- Modify: `crates/yserver-core/src/nested.rs`

### Wire formats

**GrabButton (opcode 28):**
- `header.data` = `button` (0 = AnyButton)
- body bytes 0–3: `grab_window` (u32)
- body bytes 4–5: `event_mask` (u16, `SETofPOINTEREVENT`)
- body byte 6: `pointer_mode` (0=Sync, 1=Async)
- body byte 7: `keyboard_mode` (0=Sync, 1=Async)
- body bytes 8–11: `confine_to` (u32, window or 0)
- body bytes 12–15: `cursor` (u32, cursor or 0)
- body byte 16: `button` (same as header.data)
- body byte 17: unused
- body bytes 18–19: `modifiers` (u16, 0x8000 = AnyModifier)

**UngrabButton (opcode 29):**
- `header.data` = `button`
- body bytes 0–3: `grab_window` (u32)
- body bytes 4–5: `modifiers` (u16)

**AllowEvents (opcode 35):**
- `header.data` = `mode` (0=AsyncPointer, 1=SyncPointer, 2=ReplayPointer, 3=AsyncKeyboard, 4=SyncKeyboard, 5=ReplayKeyboard, 6=AsyncBoth, 7=SyncBoth)
- body bytes 0–3: `time` (u32, timestamp)

- [ ] **Step 8: Add `find_passive_grab` method to `ServerState`**

In `server.rs`:

```rust
impl ServerState {
    /// Walk from `window` up to root looking for a matching passive button grab.
    /// Returns a clone of the first match found (innermost window wins).
    #[must_use]
    pub fn find_passive_grab(
        &self,
        window: ResourceId,
        button: u8,
        state_mask: u16,
    ) -> Option<PassiveButtonGrab> {
        let mut current = window;
        let mut depth = 0usize;
        loop {
            for grab in &self.button_grabs {
                if grab.grab_window != current {
                    continue;
                }
                let button_match = grab.button == 0 || grab.button == button;
                let mod_match = grab.modifiers == 0x8000
                    || (grab.modifiers & 0xff) == (state_mask & 0xff) as u16;
                if button_match && mod_match {
                    return Some(grab.clone());
                }
            }
            let w = self.resources.window(current)?;
            if w.parent == current || w.parent == crate::resources::ROOT_WINDOW {
                break;
            }
            current = w.parent;
            depth += 1;
            if depth > 256 {
                break;
            }
        }
        None
    }
}
```

```rust
pub fn pointer_event_fanout_ungrabbed(
    state: &Mutex<ServerState>,
    event: &crate::host_x11::HostPointerEvent,
) {
    // Same as the non-grab path of pointer_event_fanout; shared via a helper.
    // For now, call pointer_event_fanout with pointer_grab already cleared.
    // The grab was already cleared by AllowEvents before this is called.
    // Use a dummy xid_map by re-routing through a simplified path.
    let top_level_id = {
        // We need the xid_map here but don't have it. Store it in ServerState.
        // See below — add xid_map_snapshot to ServerState for replay.
        match state.lock() {
            Ok(s) => s.replay_xid_map.as_ref().and_then(|m| m.get(&event.host_xid).copied()),
            Err(_) => return,
        }
    };
    let Some(top_level_id) = top_level_id else { return };
    let mask_bit: u32 = match event.kind {
        crate::host_x11::PointerEventKind::ButtonPress => 0x0000_0004,
        crate::host_x11::PointerEventKind::ButtonRelease => 0x0000_0008,
        crate::host_x11::PointerEventKind::MotionNotify => 0x0000_0040,
        crate::host_x11::PointerEventKind::EnterNotify => 0x0000_0010,
        crate::host_x11::PointerEventKind::LeaveNotify => 0x0000_0020,
    };
    use yserver_protocol::x11::{self, SequenceNumber};
    use std::sync::atomic::Ordering;
    let (nested_id, event_x, event_y, targets) = match state.lock() {
        Ok(g) => {
            let (target, ex, ey) = g
                .resources
                .pointer_target_at(top_level_id, event.event_x, event.event_y)
                .unwrap_or((top_level_id, event.event_x, event.event_y));
            let mut targets = g.subscribers(target, mask_bit);
            if targets.is_empty() && target != top_level_id {
                targets = g.subscribers(top_level_id, mask_bit);
            }
            (target, ex, ey, targets)
        }
        Err(_) => return,
    };
    for target in targets {
        let seq = SequenceNumber(target.last_sequence.load(Ordering::Relaxed));
        let mut buf = Vec::with_capacity(32);
        match event.kind {
            crate::host_x11::PointerEventKind::ButtonPress => x11::encode_button_press_event(
                &mut buf,
                target.byte_order,
                x11::PointerEvent {
                    sequence: seq,
                    detail: event.detail,
                    time: event.time,
                    root: crate::resources::ROOT_WINDOW,
                    event: nested_id,
                    root_x: event.root_x,
                    root_y: event.root_y,
                    event_x,
                    event_y,
                    state: event.state,
                },
            ),
            _ => continue,
        }
        if let Ok(mut w) = target.writer.lock() {
            let _ = w.write_all(&buf);
        }
    }
}
```

## Task 2: Drawing completeness — FillPoly, PolyText16, CopyGC, PolyPoint

fvwm3 and its modules use FillPoly for decoration fills, PolyText16 for some text,
and CopyGC to clone a base GC. Without these, decorations may be partially invisible
or missing text.

**Files:**
- Modify: `crates/yserver-core/src/nested.rs`
- Modify: `crates/yserver-core/src/host_x11.rs`
- Modify: `crates/yserver-protocol/src/x11/mod.rs`

### Wire formats

**FillPoly (opcode 69):**
- body bytes 0–3: `drawable` (u32)
- body bytes 4–7: `gc` (u32)
- body byte 8: `shape` (0=Complex, 1=Nonconvex, 2=Convex)
- body byte 9: `coord_mode` (0=Origin, 1=Previous)
- body bytes 10–11: unused
- body bytes 12+: list of `POINT = i16, i16` pairs

**PolyText16 (opcode 75):**
Identical layout to PolyText8 (opcode 74): `drawable(4) gc(4) x(i16) y(i16)` then
text items. Forward the raw body to host in the same way `poly_text8` does.

**CopyGC (opcode 57):**
- body bytes 0–3: `src_gc` (u32)
- body bytes 4–7: `dst_gc` (u32)
- body bytes 8–11: `value_mask` (u32) — which GC attributes to copy

**PolyPoint (opcode 64):**
- body bytes 0–3: `drawable` (u32)
- body bytes 4–7: `gc` (u32)
- body bytes 8+: list of `POINT = i16, i16` pairs

- [ ] **Step 1: Add `fill_poly` to `HostX11` in `host_x11.rs`**

Pattern: look at `poly_line` — it calls `XDrawLines`. `XFillPolygon` is the
corresponding Xlib call. Add after `poly_line`:

```rust
pub fn fill_poly(
    &mut self,
    host_xid: u32,
    foreground: u32,
    coord_mode: u8,
    points: &[u8],
) -> io::Result<()> {
    if points.len() < 4 {
        return Ok(());
    }
    let npoints = (points.len() / 4) as i32;
    // Reinterpret the byte slice as XPoint array via raw pointer.
    // SAFETY: XPoint is repr(C) { short x; short y; } matching our 4-byte LE pairs.
    let xpoints = points.as_ptr() as *const x11::XPoint;
    unsafe {
        x11::XSetForeground(self.display, self.gc, foreground as u64);
        x11::XFillPolygon(
            self.display,
            host_xid as u64,
            self.gc,
            xpoints,
            npoints,
            x11::Complex as i32,
            if coord_mode == 0 { x11::CoordModeOrigin } else { x11::CoordModePrevious },
        );
        x11::XFlush(self.display);
    }
    Ok(())
}
```

```rust
pub fn copy_gc(&mut self, src: ResourceId, dst: ResourceId, value_mask: u32) {
    // Read src fields first (borrow ends before mut borrow of dst).
    let src_data = self.gcs.get(&src.0).map(|g| {
        (g.foreground, g.background, g.line_width, g.font, g.clip_rectangles.clone())
    });
    let Some((fg, bg, lw, font, clip)) = src_data else { return };
    let Some(dst_gc) = self.gcs.get_mut(&dst.0) else { return };
    if value_mask & (1 << 3) != 0 { dst_gc.foreground = fg; }
    if value_mask & (1 << 4) != 0 { dst_gc.background = bg; }
    if value_mask & (1 << 8) != 0 { dst_gc.line_width = lw; }
    if value_mask & (1 << 14) != 0 { dst_gc.font = font; }
    if value_mask & (1 << 22) != 0 { dst_gc.clip_rectangles = clip; }
}
```

Value mask bit positions (from X11 spec):
- bit 3 = foreground
- bit 4 = background
- bit 8 = line-width
- bit 14 = font
- bit 22 = clip-mask (ClipRectangles)

```rust
/// Returns the screen-absolute (x, y) of the window's top-left corner.
/// Returns (0, 0) for the root window.
pub fn window_absolute_position(&self, id: ResourceId) -> (i32, i32) {
    if id == ROOT_WINDOW {
        return (0, 0);
    }
    let mut ax: i32 = 0;
    let mut ay: i32 = 0;
    let mut current = id;
    let mut depth = 0usize;
    while current != ROOT_WINDOW && depth < 256 {
        let Some(w) = self.windows.get(&current.0) else { break };
        ax += i32::from(w.x);
        ay += i32::from(w.y);
        if w.parent == current { break }
        current = w.parent;
        depth += 1;
    }
    (ax, ay)
}

/// Returns the child of `parent` that contains the point (abs_x, abs_y) in
/// screen coordinates, or `None` if no mapped child contains it.
pub fn child_at_absolute(&self, parent: ResourceId, abs_x: i32, abs_y: i32) -> Option<ResourceId> {
    let (px, py) = self.window_absolute_position(parent);
    let local_x = abs_x - px;
    let local_y = abs_y - py;
    let w = self.windows.get(&parent.0)?;
    // Walk children in reverse (top of stacking order first).
    for child_id in w.children.iter().rev().copied() {
        let Some(child) = self.windows.get(&child_id.0) else { continue };
        if child.map_state == MapState::Unmapped { continue }
        let cx = i32::from(child.x);
        let cy = i32::from(child.y);
        let cw = i32::from(child.width);
        let ch = i32::from(child.height);
        if local_x >= cx && local_x < cx + cw && local_y >= cy && local_y < cy + ch {
            return Some(child_id);
        }
    }
    None
}
```

```rust
    sequence: SequenceNumber,
    atoms: &[AtomId],
```

```rust
            let s = lock_server(server)?;
            s.selection_owner_target(selection)
        };
        if let Some((owner_window, owner_target)) = owner_info {
            let seq = SequenceNumber(owner_target.last_sequence.load(std::sync::atomic::Ordering::Relaxed));
            let mut buf = Vec::with_capacity(32);
            x11::encode_selection_request_event(
                &mut buf,
                seq,
                owner_target.byte_order,
                time,
                owner_window,
                requestor,
                selection,
                target,
                property,
            );
            if let Ok(mut w) = owner_target.writer.lock() {
                let _ = w.write_all(&buf);
            }
        } else {
            // No owner: send SelectionNotify with property=None directly to requestor.
            // Find the requestor's client.
            let requestor_target = {
                let s = lock_server(server)?;
                let owner = s.resources.window_owner(requestor)?;
                s.client_target(owner)
            };
            if let Some(rt) = requestor_target {
                // Encode SelectionNotify (type 31) inline:
                let seq = SequenceNumber(rt.last_sequence.load(std::sync::atomic::Ordering::Relaxed));
                let mut buf = [0u8; 32];
                buf[0] = 31; // SelectionNotify
                buf[2] = (seq.0 & 0xff) as u8;
                buf[3] = ((seq.0 >> 8) & 0xff) as u8;
                buf[4..8].copy_from_slice(&time.to_le_bytes());
                buf[8..12].copy_from_slice(&requestor.0.to_le_bytes());
                buf[12..16].copy_from_slice(&selection.0.to_le_bytes());
                buf[16..20].copy_from_slice(&target.0.to_le_bytes());
                // property = 0 (None) — conversion failed
                if let Ok(mut w) = rt.writer.lock() {
                    let _ = w.write_all(&buf);
                }
            }
        }
    }
    Ok(())
}
```
